# PostCSS Local Scope Example

A small working example of [postcss-local-scope].

This project is most notably using the following tools:

 - [Webpack]
 - [css-loader]
 - [PostCSS] and [postcss-loader]
 - [postcss-local-scope]

## Run the example

```bash
$ npm install
$ npm start & open http://localhost:8080
```
